About The Project

This website is to function as a sort of bread cataloging website that doubles as a place you could order bread from. The discussion page is for people to leave their thoughts and feelings about the various bread types and the index page acts as a place to see what's new on the website. The comment function currently only locally adds comments that disappear on reload, but it's my understanding that things we learn later will allow us to permanently add them for the final project.

Jacob Ackerman and Ricky Quiroga 2019
